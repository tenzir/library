The Splunk package now maps additional OCSF 1.8 event classes to Splunk CIM
datasets via `splunk::cim::map`, including Network Traffic, Web,
Authentication, Endpoint Processes and Filesystem, Change, Email, Network
Sessions VPN, Alerts, Vulnerabilities, Data Access, and Database Query shapes.
